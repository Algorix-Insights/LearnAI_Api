# BackEnd Docs

Documentacion tecnica del repositorio backend de LearnIA.

## Contenido esperado
- Arquitectura del backend.
- Contratos de API.
- Reglas de dominio.
- Decisiones tecnicas y ADRs.
- Flujos de ingesta, RAG y generacion.
- Reglas de seguridad, observabilidad y despliegue.

## Sugerencia de estructura
- `arquitectura/`: diagramas y explicacion de capas.
- `adr/`: decisiones de arquitectura.
- `api/`: contratos y ejemplos de endpoints.
- `seguridad/`: RLS, auth, validaciones y hardening.
- `operaciones/`: runbooks, despliegue y monitoreo.

## Nota
La documentacion debe vivir junto al repo para versionarse con el codigo y reflejar cambios del backend sin depender del workspace raiz.
